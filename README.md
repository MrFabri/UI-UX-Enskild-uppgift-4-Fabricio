# GET

## `/api/movies`

> Returns all movies

## `/api/movies/{id}`

> Returns a movie

## `/api/movies/{id}/screenings`

> Returns all the screenings for a movie

## `/api/screenings`

> Returns all screenings

## `/api/screenings/startpage`

> Returns screenings for the next 5 days and maximum 10 screenings.

## `/api/reviews`

> Returns all reviews

## `/api/reviews/{id}`

> Returns reviews for a movie

# POST
